## CloudFlare דואר זבל אנשים
</summary>


Cloudflare שולח דוא"ל ספאם למשתמשים שאינם Cloudflare.

- שלח דוא"ל רק למנויים שהצטרפו
- כאשר המשתמש אומר "עצור", הפסיק לשלוח דוא"ל

זה כזה פשוט. אבל לענן זה לא אכפת.
Cloudflare מסרה כי השימוש בשירותם [יכול לעצור את כל הדוארסים או התוקפים](https://support.cloudflare.com/hc/en-us/articles/200170066-Will-activating-Cloudflare-stop-all-spammers-or-attackers- ).
איך נוכל לעצור _Cloudflare spammers_ מבלי להפעיל Cloudflare?

## אפליה של ספק הדפדפן
</summary>


CloudFlare מעניקה יחס מועדף לאלה המשתמשים בפיירפוקס תוך מתן יחס עוין למשתמשים שאינם טור-דפדפן על פני טור.
משתמשי טור אשר מסרבים בצדק לבצע JavaScript לא בחינם זוכים לטיפול עוין.
אי-שוויון גישה זה הוא ניצול לרעה של ניטרליות ברשת וניצול לרעה של כוח.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browdifftbcx.gif)

- משמאל: `דפדפן טור`, מימין:` כרום`. אותה כתובת IP.

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browserdiff.jpg)

- משמאל: `[דפדפן Tor] Javascript מושבת, Cookie מופעל`
- מימין: `[Chrome] JavaScript מאופשר, קובץ ה- Cookie מושבת`

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfsiryoublocked.jpg)

- QuteBrowser (דפדפן מינורי) ללא Tor (IP של Clearnet)

| *** דפדפן *** | *** טיפול בגישה *** |
| --- | --- |
| דפדפן טור (Javascript מופעל) | גישה מותרת |
| Firefox (Javascript מופעל) | גישה מושפלת |
| כרום (Javascript מופעל) | גישה מושפלת (דוחף את גוגל reCAPTCHA) |
| כרום או פיירפוקס (JavaScript מושבת) | הגישה נדחתה (דוחף * שבור * reCAPTCHA של גוגל) |
| כרום או פיירפוקס (קובץ ה- cookie מושבת) | הגישה נדחתה |
| QuteBrowser | הגישה נדחתה |
| לינקס | הגישה נדחתה |
| w3m | הגישה נדחתה |
| wget | הגישה נדחתה |


"_מדוע לא להשתמש בכפתור השמע כדי לפתור אתגר קל? _"

כן, יש כפתור שמע, אך _ _ תמיד_ [לא עובד מעל טור](https://trac.torproject.org/projects/tor/ticket/23840). תקבל הודעה זו כשתלחץ עליה:

```
נסה שוב מאוחר יותר
ייתכן שהמחשב או הרשת שלך שולחים שאילתות אוטומטיות.
כדי להגן על המשתמשים שלנו, איננו יכולים לעבד את בקשתך ברגע זה.
לפרטים נוספים בקר בדף העזרה שלנו
``

## דיכוי בוחרים
</summary>


מצביעים במדינות ארה"ב נרשמים להצביע בסופו של דבר דרך אתר מזכיר המדינה במדינת מגוריהם.
משרדי מזכירות המדינה בשליטת הרפובליקנים עוסקים בדיכוי המצביעים על ידי הקרנת אתר מזכיר המדינה באמצעות Cloudflare.
הטיפול העוין של Cloudflare במשתמשי Tor, עמדת ה- MITM שלה כנקודת מעקב גלובלית ריכוזית, ותפקידה המזיק בכלל
גורם לבוחרים פוטנציאליים להימנע מלהירשם. ליברלים בפרט נוטים לאמץ פרטיות. טפסי הרשמה של בוחרים אוספים מידע רגיש אודות נטייתו הפוליטית של הבוחר, כתובת פיזית אישית, מספר תעודת זהות ותאריך לידה.
מרבית המדינות רק מייצגות תת-מידע של המידע הזה, אך Cloudflare רואה את כל המידע הזה כאשר מישהו נרשם להצביע.

שים לב כי רישום הנייר אינו עוקף את Cloudflare מכיוון שמזכיר עובדי הזנת נתוני המדינה ככל הנראה ישתמש ב-
אתר Cloudflare להזנת הנתונים.

- Change.org הוא אתר מפורסם לאיסוף קולות ולנקיטת פעולות. "[אנשים בכל מקום פותחים בקמפיינים, מגייסים תומכים ופועלים עם מקבלי החלטות כדי להניע פתרונות.](https://web.archive.org/web/20200206120027/https://www.change.org/about)"
לרוע המזל, אנשים רבים אינם יכולים לצפות ב-.org בכלל בגלל המסנן האגרסיבי של Cloudflare. הם נחסמים מלחתום על העצומה, ובכך להוציא אותם מהליך דמוקרטי. שימוש בפלטפורמה אחרת שאינה מעוננת בענן, כגון [OpenPetition](https://www.openpetition.eu/content/about_us) עוזר לתקן את הבעיה.

## שקר לגבי מחיקת נתוני המשתמש
</summary>


על פי [הבלוג של הלקוח לשעבר ב- Cloudflare](https://shkspr.mobi/blog/2019/11/can-you-trust-cloudflare-with-your-personal-data/), Cloudflare משקרת על מחיקת חשבונות. בימינו, הרבה [חברות שומרות על הנתונים שלך](https://justdeleteme.xyz/) לאחר שסגרת או הסרת את חשבונך. רוב החברות הטובות אכן מזכירות זאת במדיניות הפרטיות שלהן. פרח ענן? לא.

```
2019-08-05 CloudFlare שלחה לי אישור שהסירו את חשבוני.
2019-10-02 קיבלתי דוא"ל מ- CloudFlare "כי אני לקוח"
```

Cloudflare לא ידע על המילה "הסר". אם זה באמת הוסר, מדוע לקוח לשעבר הזה קיבל דוא"ל? הוא ציין כי מדיניות הפרטיות של Cloudflare אינה מציינת זאת.

```
מדיניות הפרטיות החדשה שלהם לא מזכירה את שמירת הנתונים למשך שנה.
```

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfviopl_notdel.jpg)

איך אתה יכול לסמוך על Cloudflare אם [מדיניות הפרטיות שלהם היא LIE](https://twitter.com/daviddlow/status/1197787135526555648)?

``
הגיש כרטיס תמיכה באמצעות הקטגוריה "חשבון",
ולבקש מחיקת חשבון בגוף ההודעות.
אסור שיהיו לך תחומים או כרטיסי אשראי מחוברים לחשבונך לפני שתבקש למחוק.
```

אתה תקבל [דוא"ל אישור זה](https://twitter.com/originalesushi/status/1199041528414527495).
