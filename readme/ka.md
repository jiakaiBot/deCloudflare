|  Cloudflare рЃљрЃарЃўрЃА рЃЏрЃАрЃЮрЃцрЃџрЃўрЃЮрЃерЃў рЃБрЃЊрЃўрЃЊрЃћрЃАрЃў MITM рЃЏрЃљрЃарЃўрЃЮрЃюрЃћрЃбрЃБрЃџрЃў (рЃАрЃљрЃърЃўрЃарЃўрЃАрЃърЃўрЃарЃЮ рЃЏрЃљрЃарЃўрЃЮрЃюрЃћрЃбрЃБрЃџрЃў).Cloudflare рЃцрЃџрЃЮрЃЉрЃА CDN рЃЉрЃљрЃќрЃарЃўрЃА 80% -рЃќрЃћ рЃЏрЃћрЃб рЃгрЃўрЃџрЃА рЃЊрЃљ cloudflare рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ рЃДрЃЮрЃЋрЃћрЃџрЃЊрЃдрЃўрЃБрЃарЃљрЃЊ рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ.рЃЏрЃљрЃЌ рЃњрЃљрЃљрЃцрЃљрЃарЃЌрЃЮрЃЋрЃћрЃА рЃЦрЃАрЃћрЃџрЃў рЃљрЃАрЃќрЃћ рЃЏрЃћрЃб рЃЦрЃЋрЃћрЃДрЃљрЃюрЃљрЃерЃў.Cloudflare рЃБрЃцрЃарЃЮ рЃЏрЃћрЃб рЃЋрЃћрЃЉ РђЊ рЃбрЃарЃљрЃцрЃўрЃЎрЃА рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ рЃЋрЃўрЃЊрЃарЃћ Twitter, Amazon, Apple, Instagram, Bing рЃЊрЃљ WikipediaРђЊ рЃўрЃА рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃўрЃарЃћрЃЉрЃБрЃџрЃў.Cloudflare рЃњрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃЌ рЃБрЃцрЃљрЃАрЃЮ рЃњрЃћрЃњрЃЏрЃљрЃА рЃЊрЃљ рЃЉрЃћрЃЋрЃарЃў рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЏрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃўрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ.рЃўрЃАрЃўрЃюрЃў рЃЎрЃЮрЃюрЃцрЃўрЃЊрЃћрЃюрЃфрЃўрЃљрЃџрЃБрЃарЃЮрЃЉрЃљрЃќрЃћ рЃЋрЃљрЃГрЃарЃЮрЃЉрЃЊрЃюрЃћрЃю рЃЎрЃЮрЃюрЃцрЃўрЃЊрЃћрЃюрЃфрЃўрЃљрЃџрЃБрЃарЃЮрЃЉрЃљрЃА.  |  ![](../image/cfmarketshare.jpg)  |
|  Cloudflare рЃќрЃўрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃАрЃљ рЃЊрЃљ рЃгрЃљрЃарЃЏрЃЮрЃерЃЮрЃЉрЃўрЃА рЃЋрЃћрЃЉ рЃАрЃћрЃарЃЋрЃћрЃарЃА рЃерЃЮрЃарЃўрЃА, рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃА рЃАрЃљрЃАрЃљрЃќрЃдрЃЋрЃарЃЮ рЃАрЃљрЃърЃљрЃбрЃарЃБрЃџрЃЮ рЃљрЃњрЃћрЃюрЃбрЃўрЃА рЃЏрЃАрЃњрЃљрЃЋрЃАрЃљрЃЊ.рЃЌрЃЦрЃЋрЃћрЃю рЃЋрЃћрЃа рЃерЃћрЃФрЃџрЃћрЃЉрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃљрЃарЃЕрЃћрЃБрЃџ рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃА рЃљрЃЊрЃњрЃўрЃџрЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљрЃА.рЃЌрЃЦрЃЋрЃћрЃю рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃўрЃЌ Cloudflare- рЃА рЃЊрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃДрЃЋрЃћрЃџрЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃерЃўрЃцрЃЋрЃарЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃАрЃфрЃћрЃЏрЃћрЃю рЃцрЃарЃћрЃюрЃљрЃА. |  рЃгрЃљрЃарЃЏрЃЮрЃерЃЮрЃЉрЃўрЃА рЃЋрЃћрЃЉ РђЊ рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃЏрЃљ рЃљрЃњрЃћрЃюрЃбрЃА - CloudflareРђЊ рЃА рЃЏрЃўрЃАрЃфрЃљ рЃБрЃцрЃџрЃћрЃЉрЃљ, рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃўрЃбрЃЮрЃА, рЃЋрЃўрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃерЃћрЃАрЃЋрЃџрЃљ рЃЏрЃљрЃЌ РђърЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃАРђю рЃЊрЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃЮрЃА РђърЃерЃћрЃќрЃдрЃБрЃЊрЃБрЃџрЃў рЃбрЃћрЃарЃўрЃбрЃЮрЃарЃўрЃљРђю.  |  ![](../image/usershoulddecide.jpg)  |
|  рЃњрЃљрЃЊрЃљрЃ«рЃћрЃЊрЃћрЃЌ рЃАрЃгрЃЮрЃа рЃАрЃБрЃарЃљрЃЌрЃА.рЃЌрЃЦрЃЋрЃћрЃю рЃўрЃцрЃўрЃЦрЃарЃћрЃЉрЃЌ Cloudflare рЃЉрЃџрЃЮрЃЎрЃљрЃЋрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃфрЃБрЃЊрЃў рЃЉрЃўрЃГрЃћрЃЉрЃўрЃА.рЃЌрЃЦрЃЋрЃћрЃю рЃўрЃцрЃўрЃЦрЃарЃћрЃЉрЃЌ, рЃарЃЮрЃЏ Cloudflare рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃЮрЃюрЃџрЃљрЃўрЃюрЃерЃўрЃљ (рЃљрЃарЃљрЃАрЃЊрЃарЃЮрЃА рЃЊрЃљрЃћрЃџрЃЮрЃЊрЃЮрЃЌ).рЃЌрЃЦрЃЋрЃћрЃю рЃўрЃцрЃўрЃЦрЃарЃћрЃЉрЃЌ рЃџрЃћрЃњрЃўрЃбрЃўрЃЏрЃБрЃа рЃЉрЃЮрЃбрЃћрЃЉрЃА рЃЊрЃљ рЃЏрЃфрЃЮрЃфрЃљрЃЋрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃўрЃарЃћрЃЉрЃљ.  |  ![](../image/howcfwork.jpg)  |
|  рЃЌрЃБрЃЏрЃфрЃљ рЃћрЃА рЃАрЃљрЃћрЃарЃЌрЃЮрЃЊ рЃљрЃа рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ рЃАрЃўрЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃћрЃА.Cloudflare рЃБрЃерЃћрЃфрЃЊрЃЮрЃЏрЃЮрЃЊ рЃљрЃЎрЃарЃФрЃљрЃџрЃљрЃЋрЃА рЃБрЃЊрЃљрЃюрЃљрЃерЃљрЃБрЃџрЃЮ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃћрЃЉрЃА.Cloudflare рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЕрЃљрЃЋрЃўрЃЊрЃћрЃА.Cloudflare рЃЉрЃџрЃЮрЃЎрЃљрЃЋрЃА рЃџрЃћрЃњрЃћрЃюрЃЊрЃБрЃџ рЃЉрЃЮрЃбрЃћрЃЉрЃА.  |  ![](../image/cfdowncfcom.jpg)  |
|  рЃўрЃАрЃћрЃЋрЃћ, рЃарЃЮрЃњрЃЮрЃарЃф рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃ░рЃЮрЃАрЃбрЃўрЃюрЃњрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃў, Cloudflare рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃарЃБрЃџрЃДрЃЮрЃцрЃўрЃџрЃў.рЃЌрЃЦрЃЋрЃћрЃю рЃюрЃљрЃ«рЃљрЃЋрЃЌ рЃљрЃЏ рЃћрЃЎрЃарЃљрЃюрЃА рЃЏрЃљрЃерЃўрЃюрЃљрЃф рЃЎрЃў, рЃЌрЃБ рЃгрЃљрЃарЃЏрЃЮрЃерЃЮрЃЉрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃЎрЃљрЃарЃњрЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА.  |  ![](../image/cfdown2019.jpg) |
