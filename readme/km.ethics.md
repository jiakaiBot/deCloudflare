## CloudFlare បោកប្រាស់មនុស្ស
</summar>


Cloudflare កំពុងផ្ញើអ៊ីម៉ែលសារឥតបានការទៅកាន់អ្នកប្រើប្រាស់ដែលមិនមែនជាហ្វាលហ្វារ។

ផ្ញើតែអ៊ីមែលទៅអតិថិជនដែលបានជ្រើសរើស
- នៅពេលអ្នកប្រើប្រាស់និយាយថា "បញ្ឈប់" បន្ទាប់មកឈប់ផ្ញើអ៊ីមែល

វាសាមញ្ញណាស់។ ប៉ុន្តែ Cloudflare មិនខ្វល់ទេ។
Cloudflare បាននិយាយថាការប្រើប្រាស់សេវាកម្មរបស់ពួកគេ [អាចបញ្ឈប់រាល់អ្នកផ្ញើសារឥតបានការរឺអ្នកវាយប្រហារ](https://support.cloudflare.com/hc/en-us/articles/200170066-Will-activating-Cloudflare-stop-all-spammers-or-attackers- ) ។
តើយើងអាចបញ្ឈប់ _Cloudflare spammers_ ដោយមិនធ្វើឱ្យ Cloudflare សកម្មយ៉ាងដូចម្តេច?

## ការរើសអើងអ្នកលក់កម្មវិធីរុករក
</summar>


CloudFlare ផ្តល់នូវការអនុគ្រោះដល់អ្នកដែលប្រើប្រាស់ Firefox ខណៈពេលដែលផ្តល់ការប្រទូសរ៉ាយដល់អ្នកប្រើប្រាស់ដែលមិនមែនជា Tor-Browser លើ Tor ។
អ្នកប្រើតោរបស់អ្នកដែលបដិសេធក្នុងការប្រតិបត្តិសៀវភៅដោយឥតគិតថ្លៃក៏ទទួលការព្យាបាលដោយអរិភាពដែរ។
វិសមភាពក្នុងការចូលប្រើនេះគឺជាការបំពានអព្យាក្រឹតភាពបណ្តាញនិងការរំលោភអំណាច។

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browdifftbcx.gif)

- ឆ្វេង៖ `តោនរុករកអ៊ីនធឺរ` ខាងស្តាំៈ Chrome ។ អាសយដ្ឋាន IP ដូចគ្នា។

![](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browserdiff.jpg)

- ខាងឆ្វេង៖ `[តូប្រូសឺរ] ចាវ៉ាស្គ្រីបបិទខូឃីបានបើក`
- សិទ្ធិ៖ `[Chrome] Javascript បានបើកដំណើរការខូឃីបានបិទ`
