## CloudFlare jendeak zabaltzen du
</Laburpen>


Cloudflare-k spam mezu elektronikoak bidaltzen ditu Cloudflare ez diren erabiltzaileei.

- Bidali mezu elektronikoak soilik aukeran parte hartu duten harpidedunei

Sinplea da. Baina Cloudflarek ez du axola.
Cloudflarek esan du bere zerbitzua erabiliz [iruzur edo erasotzaile guztiak geldiaraztea](https://support.cloudflare.com/hc/en-us/articles/200170066-Will-activating-Cloudflare-stop-all-spammers-or-attackers- ).

## Arakatzailearen saltzaileen bereizkeria
</Laburpen>


CloudFlare-k lehentasunezko tratamendua ematen die Firefox erabiltzen dutenei Tor-arakatzaile ez diren erabiltzaileei Tor-en aurkako tratamendu etsaia ematen dieten bitartean.
Doakoak ez diren Javascriptak exekutatzeko eskubidea duten erabiltzaileek ere tratamendu etsaiak jasotzen dituzte.
Sarbide desberdintasuna sareko neutraltasun gehiegikeria eta botere gehiegikeria da.

[](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browdifftbcx.gif)

Ezkerrean: `Tor Browser`, eskuinean:` Chrome`. IP helbide bera.

[](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/browserdiff.jpg)

- Ezkerrean: `[Tor Arakatzailea] Javascript desgaituta, cookie gaituta '
Eskuinean: `[Chrome] Javascript gaituta, cookie desgaituta '

[](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfsiryoublocked.jpg)

- QuteBrowser (arakatzaile txikia) Tor gabe (Clearnet IP)

| *** Arakatzailea *** | *** Sarbide tratamendua *** |
| --- | --- |
| Tor Arakatzailea (Javascript gaituta) | sarbidea baimenduta |
| Firefox (Javascript gaituta) | sarbidea degradatua |
| Chromium (Javascript gaituta) | sarbidea degradatua (Google reCAPTCHA bultzatzen du) |
| Chromium edo Firefox (Javascript desgaituta) | sarbidea ukatu (bultzaka * hautsita * Google reCAPTCHA) |
| Chromium edo Firefox (Cookie desgaituta) | sarbidea ukatu |
| QuteBrowser | sarbidea ukatu |
| katamotza | sarbidea ukatu |
| w3m | sarbidea ukatu |
| wget | sarbidea ukatu |


"_Zergatik ez duzu audio botoia erabiltzen erronka erraza konpontzeko? _"

Bai, audio botoia dago, baina _always_-ek [ez du Tor-en bidez funtzionatzen](https://trac.torproject.org/projects/tor/ticket/23840). Mezu hau klikatzen duzunean jasoko duzu:

```
Saiatu berriro geroago
Baliteke zure ordenagailuak edo sareak kontsulta automatikoak bidaltzen dituela.
Gure erabiltzaileak babesteko, ezin dugu zure eskaera oraintxe prozesatu.
Xehetasun gehiago nahi izanez gero, bisitatu gure laguntza orria
```

## Botoa ezabatzea
</Laburpen>


AEBetako estatuetako hautesleek, azken finean, egoitzako estatu idazkariaren webgunearen bidez bozkatzeko erregistratzen dira.
Errepublikanoek kontrolatutako estatuko idazkariak hautesleen ezabapenean aritzen dira Cloudflare bidez estatu idazkariaren webgunea proxy bidez.
Cloud erabiltzaileek Tor erabiltzaileen tratamendu etsaia, MITM posizioa zaintza puntu global gisa eta bere zeregin orokorra
Hautesle potentzialek erregistratzeari uko egiten diote. Liberalek bereziki, pribatutasuna besarkatzen dute. Boto-emaileen erregistro-formularioek hautesleen agur politikoari, helbide fisiko pertsonalari, gizarte segurantzako zenbakia eta jaiotze datari buruzko informazio sentikorra biltzen dute.
Estatu gehienek informazio horren azpimultzo bat besterik ez dute jendaurrean eskuragarri, baina Cloudflarek *** informazio guztia ikusten du norbaitek botoa erregistratzen duenean.

Kontuan izan paperak erregistratzeak ez duela Cloudflare saihesten, datuen sarrerako langileen idazkari nagusiak seguraski erabiliko dituelako
Cloudflare webgunea datuak sartzeko.

[](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfvotm_01.jpg)
[](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/cfvotm_02.jpg)

- Change.org webgune ospetsua da botoak bildu eta neurriak hartzeko. "[jendea nonahi ari da kanpainak egiten, aldekoak mobilizatzen eta erabakiak hartzen dituztenekin lan egiten soluzioak bultzatzeko.](https://web.archive.org/web/20200206120027/https://www.change.org/about)"
Zoritxarrez, jende askok ezin du inolaz ere ikusi change.org Cloudflareren iragazki oldarkorra dela eta. Eskaera sinatzen ari dira blokeatuta, eta horrela, prozesu demokratiko batetik baztertzen dituzte. Hodeirik gabeko beste plataforma batzuk erabiltzeak, esaterako, [OpenPetition](https://www.openpetition.eu/content/about_us) arazoa konpontzen lagunduko du.

[](https://codeberg.org/crimeflare/cloudflare-tor/media/branch/master/image/changeorgasn.jpg)

Esan dutenez, "hautesleek informazioa hauteskundeetako informazioa eta hautesleen erregistroa_ sar dezakete", baina hori gezurra da, jende askok ezin baitu inolaz ere arakatu gunea.
