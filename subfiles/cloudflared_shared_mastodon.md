### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *1,159,035* public toots. (about *17044.6* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *473,989* links in total.
  - *118,621* links are Cloudflare links. (*25*%)
- There are *14,105* users who shared ANY links.
  - On average one mastodon user shared *33.6* links. (about *0.5* links per day)
  - There are *1,422* users who ONLY shared Cloudflare links. (*10.1*%)
  - There are *7,558* users who NEVER shared Cloudflare links. (*53.6*%)

tl;dr
### *46.4%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
