### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *354,051* public toots. (about *17702.6* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *145,274* links in total.
  - *36,645* links are Cloudflare links. (*25.2*%)
- There are *8,209* users who shared ANY links.
  - On average one mastodon user shared *17.7* links. (about *0.9* links per day)
  - There are *944* users who ONLY shared Cloudflare links. (*11.5*%)
  - There are *4,785* users who NEVER shared Cloudflare links. (*58.3*%)

tl;dr
### *41.7%* of Mastodon users who shared link shared Cloudflare links.
### *25.2%* of shared links are Cloudflare links.
