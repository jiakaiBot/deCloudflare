### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *623,851* public toots. (about *17329.2* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *250,019* links in total.
  - *62,592* links are Cloudflare links. (*25*%)
- There are *10,631* users who shared ANY links.
  - On average one mastodon user shared *23.5* links. (about *0.7* links per day)
  - There are *1,098* users who ONLY shared Cloudflare links. (*10.3*%)
  - There are *5,973* users who NEVER shared Cloudflare links. (*56.2*%)

tl;dr
### *43.8%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
