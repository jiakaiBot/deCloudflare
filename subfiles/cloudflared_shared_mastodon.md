### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *531,301* public toots. (about *17710* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *227,037* links in total.
  - *58,668* links are Cloudflare links. (*25.8*%)
- There are *9,794* users who shared ANY links.
  - On average one mastodon user shared *23.2* links. (about *0.8* links per day)
  - There are *1,042* users who ONLY shared Cloudflare links. (*10.6*%)
  - There are *5,593* users who NEVER shared Cloudflare links. (*57.1*%)

tl;dr
### *42.9%* of Mastodon users who shared link shared Cloudflare links.
### *25.8%* of shared links are Cloudflare links.
