### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *194,580* public toots. (about *17689.1* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *81,870* links in total.
  - *20,556* links are Cloudflare links. (*25.1*%)
- There are *6,264* users who shared ANY links.
  - On average one mastodon user shared *13.1* links. (about *1.2* links per day)
  - There are *738* users who ONLY shared Cloudflare links. (*11.8*%)
  - There are *3,850* users who NEVER shared Cloudflare links. (*61.5*%)

tl;dr
### *38.5%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
