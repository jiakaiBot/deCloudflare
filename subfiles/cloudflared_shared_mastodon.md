### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *440,855* public toots. (about *17634.2* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *187,665* links in total.
  - *48,027* links are Cloudflare links. (*25.6*%)
- There are *9,060* users who shared ANY links.
  - On average one mastodon user shared *20.7* links. (about *0.8* links per day)
  - There are *980* users who ONLY shared Cloudflare links. (*10.8*%)
  - There are *5,247* users who NEVER shared Cloudflare links. (*57.9*%)

tl;dr
### *42.1%* of Mastodon users who shared link shared Cloudflare links.
### *25.6%* of shared links are Cloudflare links.
