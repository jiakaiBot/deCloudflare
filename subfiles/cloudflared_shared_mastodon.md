### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *550,485* public toots. (about *17202.7* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *221,359* links in total.
  - *55,445* links are Cloudflare links. (*25*%)
- There are *10,048* users who shared ANY links.
  - On average one mastodon user shared *22* links. (about *0.7* links per day)
  - There are *1,063* users who ONLY shared Cloudflare links. (*10.6*%)
  - There are *5,692* users who NEVER shared Cloudflare links. (*56.6*%)

tl;dr
### *43.4%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
