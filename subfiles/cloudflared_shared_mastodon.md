### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *24,999* public toots. (about *12499.5* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *9,644* links in total.
  - *2,580* links are Cloudflare links. (*26.8*%)
- There are *1,890* users who shared ANY links.
  - On average one mastodon user shared *5.1* links. (about *2.6* links per day)
  - There are *270* users who ONLY shared Cloudflare links. (*14.3*%)
  - There are *1,302* users who NEVER shared Cloudflare links. (*68.9*%)

tl;dr
### *31.1%* of Mastodon users who shared link shared Cloudflare links.
### *26.8%* of shared links are Cloudflare links.
