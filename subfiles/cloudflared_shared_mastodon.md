### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *354,347* public toots. (about *17717.4* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *151,065* links in total.
  - *38,153* links are Cloudflare links. (*25.3*%)
- There are *8,185* users who shared ANY links.
  - On average one mastodon user shared *18.5* links. (about *0.9* links per day)
  - There are *909* users who ONLY shared Cloudflare links. (*11.1*%)
  - There are *4,804* users who NEVER shared Cloudflare links. (*58.7*%)

tl;dr
### *41.3%* of Mastodon users who shared link shared Cloudflare links.
### *25.3%* of shared links are Cloudflare links.
