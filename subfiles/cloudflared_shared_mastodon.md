### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *300,439* public toots. (about *17672.9* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *127,206* links in total.
  - *31,973* links are Cloudflare links. (*25.1*%)
- There are *7,598* users who shared ANY links.
  - On average one mastodon user shared *16.7* links. (about *1* links per day)
  - There are *859* users who ONLY shared Cloudflare links. (*11.3*%)
  - There are *4,513* users who NEVER shared Cloudflare links. (*59.4*%)

tl;dr
### *40.6%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
