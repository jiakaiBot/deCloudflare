### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *394,880* public toots. (about *17168.7* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *160,672* links in total.
  - *40,560* links are Cloudflare links. (*25.2*%)
- There are *8,652* users who shared ANY links.
  - On average one mastodon user shared *18.6* links. (about *0.8* links per day)
  - There are *978* users who ONLY shared Cloudflare links. (*11.3*%)
  - There are *5,025* users who NEVER shared Cloudflare links. (*58.1*%)

tl;dr
### *41.9%* of Mastodon users who shared link shared Cloudflare links.
### *25.2%* of shared links are Cloudflare links.
