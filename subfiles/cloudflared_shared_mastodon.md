### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *441,297* public toots. (about *17651.9* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *179,382* links in total.
  - *45,332* links are Cloudflare links. (*25.3*%)
- There are *9,113* users who shared ANY links.
  - On average one mastodon user shared *19.7* links. (about *0.8* links per day)
  - There are *1,003* users who ONLY shared Cloudflare links. (*11*%)
  - There are *5,249* users who NEVER shared Cloudflare links. (*57.6*%)

tl;dr
### *42.4%* of Mastodon users who shared link shared Cloudflare links.
### *25.3%* of shared links are Cloudflare links.
