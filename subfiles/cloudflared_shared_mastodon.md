### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *392,407* public toots. (about *17836.7* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *167,407* links in total.
  - *42,629* links are Cloudflare links. (*25.5*%)
- There are *8,573* users who shared ANY links.
  - On average one mastodon user shared *19.5* links. (about *0.9* links per day)
  - There are *952* users who ONLY shared Cloudflare links. (*11.1*%)
  - There are *4,993* users who NEVER shared Cloudflare links. (*58.2*%)

tl;dr
### *41.8%* of Mastodon users who shared link shared Cloudflare links.
### *25.5%* of shared links are Cloudflare links.
