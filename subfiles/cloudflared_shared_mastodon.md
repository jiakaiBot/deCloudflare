### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *515,903* public toots. (about *17789.8* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *221,102* links in total.
  - *57,013* links are Cloudflare links. (*25.8*%)
- There are *9,665* users who shared ANY links.
  - On average one mastodon user shared *22.9* links. (about *0.8* links per day)
  - There are *1,029* users who ONLY shared Cloudflare links. (*10.6*%)
  - There are *5,546* users who NEVER shared Cloudflare links. (*57.4*%)

tl;dr
### *42.6%* of Mastodon users who shared link shared Cloudflare links.
### *25.8%* of shared links are Cloudflare links.
