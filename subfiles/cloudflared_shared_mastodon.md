### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *171,748* public toots. (about *17174.8* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *71,226* links in total.
  - *17,764* links are Cloudflare links. (*24.9*%)
- There are *5,737* users who shared ANY links.
  - On average one mastodon user shared *12.4* links. (about *1.2* links per day)
  - There are *678* users who ONLY shared Cloudflare links. (*11.8*%)
  - There are *3,578* users who NEVER shared Cloudflare links. (*62.4*%)

tl;dr
### *37.6%* of Mastodon users who shared link shared Cloudflare links.
### *24.9%* of shared links are Cloudflare links.
