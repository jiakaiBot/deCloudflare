### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *146,607* public toots. (about *18325.9* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *62,458* links in total.
  - *15,613* links are Cloudflare links. (*25*%)
- There are *5,540* users who shared ANY links.
  - On average one mastodon user shared *11.3* links. (about *1.4* links per day)
  - There are *691* users who ONLY shared Cloudflare links. (*12.5*%)
  - There are *3,461* users who NEVER shared Cloudflare links. (*62.5*%)

tl;dr
### *37.5%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
