### About Mastodon Users

  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *107,937* links in total.
  - *27,024* links are Cloudflare links. (*25*%)
- There are *7,076* users who shared ANY links.
  - On average one mastodon user shared *15.3* links. (about *1* links per day)
  - There are *836* users who ONLY shared Cloudflare links. (*11.8*%)
  - There are *4,234* users who NEVER shared Cloudflare links. (*59.8*%)

### *40.2%* of Mastodon users who shared link shared Cloudflare links.
