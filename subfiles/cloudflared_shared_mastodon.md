### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *373,099* public toots. (about *17766.6* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *159,395* links in total.
  - *40,386* links are Cloudflare links. (*25.3*%)
- There are *8,366* users who shared ANY links.
  - On average one mastodon user shared *19.1* links. (about *0.9* links per day)
  - There are *927* users who ONLY shared Cloudflare links. (*11.1*%)
  - There are *4,892* users who NEVER shared Cloudflare links. (*58.5*%)

tl;dr
### *41.5%* of Mastodon users who shared link shared Cloudflare links.
### *25.3%* of shared links are Cloudflare links.
