### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *232,108* public toots. (about *17854.5* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *98,790* links in total.
  - *24,899* links are Cloudflare links. (*25.2*%)
- There are *6,808* users who shared ANY links.
  - On average one mastodon user shared *14.5* links. (about *1.1* links per day)
  - There are *794* users who ONLY shared Cloudflare links. (*11.7*%)
  - There are *4,117* users who NEVER shared Cloudflare links. (*60.5*%)

tl;dr
### *39.5%* of Mastodon users who shared link shared Cloudflare links.
### *25.2%* of shared links are Cloudflare links.
