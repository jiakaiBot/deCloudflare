### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *1,249,397* public toots. (about *17115* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *511,130* links in total.
  - *127,746* links are Cloudflare links. (*25*%)
- There are *14,527* users who shared ANY links.
  - On average one mastodon user shared *35.2* links. (about *0.5* links per day)
  - There are *1,450* users who ONLY shared Cloudflare links. (*10*%)
  - There are *7,764* users who NEVER shared Cloudflare links. (*53.4*%)

tl;dr
### *46.6%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
