### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *940,811* public toots. (about *17422.4* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *380,304* links in total.
  - *95,016* links are Cloudflare links. (*25*%)
- There are *12,872* users who shared ANY links.
  - On average one mastodon user shared *29.5* links. (about *0.5* links per day)
  - There are *1,317* users who ONLY shared Cloudflare links. (*10.2*%)
  - There are *6,995* users who NEVER shared Cloudflare links. (*54.3*%)

tl;dr
### *45.7%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
