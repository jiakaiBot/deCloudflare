### About Mastodon Users

- Mastodon users posted *178,396* public toots. (about *17839.6* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *74,691* links in total.
  - *18,704* links are Cloudflare links. (*25*%)
- There are *6,005* users who shared ANY links.
  - On average one mastodon user shared *12.4* links. (about *1.2* links per day)
  - There are *710* users who ONLY shared Cloudflare links. (*11.8*%)
  - There are *3,717* users who NEVER shared Cloudflare links. (*61.9*%)

### *38.1%* of Mastodon users who shared link shared Cloudflare links.
