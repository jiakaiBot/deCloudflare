### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *1,229,853* public toots. (about *17081.3* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *502,880* links in total.
  - *125,607* links are Cloudflare links. (*25*%)
- There are *14,435* users who shared ANY links.
  - On average one mastodon user shared *34.8* links. (about *0.5* links per day)
  - There are *1,438* users who ONLY shared Cloudflare links. (*10*%)
  - There are *7,725* users who NEVER shared Cloudflare links. (*53.5*%)

tl;dr
### *46.5%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
