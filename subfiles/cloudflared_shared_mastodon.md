### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *285,567* public toots. (about *17847.9* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *121,642* links in total.
  - *30,741* links are Cloudflare links. (*25.3*%)
- There are *7,432* users who shared ANY links.
  - On average one mastodon user shared *16.4* links. (about *1* links per day)
  - There are *837* users who ONLY shared Cloudflare links. (*11.3*%)
  - There are *4,429* users who NEVER shared Cloudflare links. (*59.6*%)

tl;dr
### *40.4%* of Mastodon users who shared link shared Cloudflare links.
### *25.3%* of shared links are Cloudflare links.
