### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *1,028,055* public toots. (about *17424.7* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *417,233* links in total.
  - *104,324* links are Cloudflare links. (*25*%)
- There are *13,394* users who shared ANY links.
  - On average one mastodon user shared *31.2* links. (about *0.5* links per day)
  - There are *1,372* users who ONLY shared Cloudflare links. (*10.2*%)
  - There are *7,224* users who NEVER shared Cloudflare links. (*53.9*%)

tl;dr
### *46.1%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
