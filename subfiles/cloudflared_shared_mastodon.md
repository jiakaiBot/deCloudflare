### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *270,052* public toots. (about *18003.5* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *115,599* links in total.
  - *29,346* links are Cloudflare links. (*25.4*%)
- There are *7,257* users who shared ANY links.
  - On average one mastodon user shared *15.9* links. (about *1.1* links per day)
  - There are *831* users who ONLY shared Cloudflare links. (*11.5*%)
  - There are *4,337* users who NEVER shared Cloudflare links. (*59.8*%)

tl;dr
### *40.2%* of Mastodon users who shared link shared Cloudflare links.
### *25.4%* of shared links are Cloudflare links.
