### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *407,988* public toots. (about *17738.6* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *173,686* links in total.
  - *44,269* links are Cloudflare links. (*25.5*%)
- There are *8,744* users who shared ANY links.
  - On average one mastodon user shared *19.9* links. (about *0.9* links per day)
  - There are *963* users who ONLY shared Cloudflare links. (*11*%)
  - There are *5,093* users who NEVER shared Cloudflare links. (*58.2*%)

tl;dr
### *41.8%* of Mastodon users who shared link shared Cloudflare links.
### *25.5%* of shared links are Cloudflare links.
