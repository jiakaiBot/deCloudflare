### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *609,627* public toots. (about *17417.9* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *244,761* links in total.
  - *61,290* links are Cloudflare links. (*25*%)
- There are *10,522* users who shared ANY links.
  - On average one mastodon user shared *23.3* links. (about *0.7* links per day)
  - There are *1,089* users who ONLY shared Cloudflare links. (*10.3*%)
  - There are *5,915* users who NEVER shared Cloudflare links. (*56.2*%)

tl;dr
### *43.8%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
