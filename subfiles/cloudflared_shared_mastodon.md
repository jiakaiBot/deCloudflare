### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *336,548* public toots. (about *17713.1* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *143,147* links in total.
  - *36,171* links are Cloudflare links. (*25.3*%)
- There are *7,987* users who shared ANY links.
  - On average one mastodon user shared *17.9* links. (about *0.9* links per day)
  - There are *888* users who ONLY shared Cloudflare links. (*11.1*%)
  - There are *4,701* users who NEVER shared Cloudflare links. (*58.9*%)

tl;dr
### *41.1%* of Mastodon users who shared link shared Cloudflare links.
### *25.3%* of shared links are Cloudflare links.
