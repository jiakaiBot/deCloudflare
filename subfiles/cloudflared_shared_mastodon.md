### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *855,511* public toots. (about *17459.4* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *344,676* links in total.
  - *85,972* links are Cloudflare links. (*24.9*%)
- There are *12,332* users who shared ANY links.
  - On average one mastodon user shared *27.9* links. (about *0.6* links per day)
  - There are *1,265* users who ONLY shared Cloudflare links. (*10.3*%)
  - There are *6,792* users who NEVER shared Cloudflare links. (*55.1*%)

tl;dr
### *44.9%* of Mastodon users who shared link shared Cloudflare links.
### *24.9%* of shared links are Cloudflare links.
