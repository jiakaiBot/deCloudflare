### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *871,225* public toots. (about *17424.5* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *350,516* links in total.
  - *87,426* links are Cloudflare links. (*24.9*%)
- There are *12,418* users who shared ANY links.
  - On average one mastodon user shared *28.2* links. (about *0.6* links per day)
  - There are *1,272* users who ONLY shared Cloudflare links. (*10.2*%)
  - There are *6,836* users who NEVER shared Cloudflare links. (*55*%)

tl;dr
### *45%* of Mastodon users who shared link shared Cloudflare links.
### *24.9%* of shared links are Cloudflare links.
