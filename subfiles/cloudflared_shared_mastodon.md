### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *1,047,071* public toots. (about *17451.2* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *426,099* links in total.
  - *106,693* links are Cloudflare links. (*25*%)
- There are *13,509* users who shared ANY links.
  - On average one mastodon user shared *31.5* links. (about *0.5* links per day)
  - There are *1,376* users who ONLY shared Cloudflare links. (*10.2*%)
  - There are *7,273* users who NEVER shared Cloudflare links. (*53.8*%)

tl;dr
### *46.2%* of Mastodon users who shared link shared Cloudflare links.
