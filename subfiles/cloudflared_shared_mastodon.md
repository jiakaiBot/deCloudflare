### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *334,081* public toots. (about *17583.2* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *137,180* links in total.
  - *34,459* links are Cloudflare links. (*25.1*%)
- There are *7,996* users who shared ANY links.
  - On average one mastodon user shared *17.2* links. (about *0.9* links per day)
  - There are *927* users who ONLY shared Cloudflare links. (*11.6*%)
  - There are *4,687* users who NEVER shared Cloudflare links. (*58.6*%)

tl;dr
### *41.4%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
