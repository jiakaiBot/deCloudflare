### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *510,801* public toots. (about *17613.8* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *206,290* links in total.
  - *52,039* links are Cloudflare links. (*25.2*%)
- There are *9,734* users who shared ANY links.
  - On average one mastodon user shared *21.2* links. (about *0.7* links per day)
  - There are *1,052* users who ONLY shared Cloudflare links. (*10.8*%)
  - There are *5,548* users who NEVER shared Cloudflare links. (*57*%)
