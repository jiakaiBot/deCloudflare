### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *713,842* public toots. (about *17410.8* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *287,446* links in total.
  - *72,199* links are Cloudflare links. (*25.1*%)
- There are *11,382* users who shared ANY links.
  - On average one mastodon user shared *25.3* links. (about *0.6* links per day)
  - There are *1,213* users who ONLY shared Cloudflare links. (*10.7*%)
  - There are *6,312* users who NEVER shared Cloudflare links. (*55.5*%)

tl;dr
### *44.5%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
