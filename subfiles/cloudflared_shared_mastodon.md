### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *569,209* public toots. (about *17248.8* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *228,862* links in total.
  - *57,305* links are Cloudflare links. (*25*%)
- There are *10,187* users who shared ANY links.
  - On average one mastodon user shared *22.5* links. (about *0.7* links per day)
  - There are *1,072* users who ONLY shared Cloudflare links. (*10.5*%)
  - There are *5,755* users who NEVER shared Cloudflare links. (*56.5*%)

tl;dr
### *43.5%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
