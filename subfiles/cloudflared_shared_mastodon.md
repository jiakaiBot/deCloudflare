### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *341,348* public toots. (about *17965.7* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *139,940* links in total.
  - *35,115* links are Cloudflare links. (*25.1*%)
- There are *8,068* users who shared ANY links.
  - On average one mastodon user shared *17.3* links. (about *0.9* links per day)
  - There are *930* users who ONLY shared Cloudflare links. (*11.5*%)
  - There are *4,721* users who NEVER shared Cloudflare links. (*58.5*%)

tl;dr
### *41.5%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
