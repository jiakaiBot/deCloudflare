### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *318,357* public toots. (about *17686.5* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *134,972* links in total.
  - *34,008* links are Cloudflare links. (*25.2*%)
- There are *7,776* users who shared ANY links.
  - On average one mastodon user shared *17.4* links. (about *1* links per day)
  - There are *885* users who ONLY shared Cloudflare links. (*11.4*%)
  - There are *4,591* users who NEVER shared Cloudflare links. (*59*%)

tl;dr
### *41%* of Mastodon users who shared link shared Cloudflare links.
### *25.2%* of shared links are Cloudflare links.
