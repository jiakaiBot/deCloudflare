### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *800,560* public toots. (about *17403.5* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *322,024* links in total.
  - *80,752* links are Cloudflare links. (*25.1*%)
- There are *11,974* users who shared ANY links.
  - On average one mastodon user shared *26.9* links. (about *0.6* links per day)
  - There are *1,245* users who ONLY shared Cloudflare links. (*10.4*%)
  - There are *6,605* users who NEVER shared Cloudflare links. (*55.2*%)

tl;dr
### *44.8%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
