### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *276,913* public toots. (about *17307.1* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *113,719* links in total.
  - *28,490* links are Cloudflare links. (*25.1*%)
- There are *7,285* users who shared ANY links.
  - On average one mastodon user shared *15.6* links. (about *1* links per day)
  - There are *856* users who ONLY shared Cloudflare links. (*11.8*%)
  - There are *4,340* users who NEVER shared Cloudflare links. (*59.6*%)

tl;dr
### *40.4%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
