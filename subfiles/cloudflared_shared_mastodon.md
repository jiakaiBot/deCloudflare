### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *223,802* public toots. (about *17215.5* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *93,776* links in total.
  - *23,442* links are Cloudflare links. (*25*%)
- There are *6,592* users who shared ANY links.
  - On average one mastodon user shared *14.2* links. (about *1.1* links per day)
  - There are *788* users who ONLY shared Cloudflare links. (*12*%)
  - There are *3,992* users who NEVER shared Cloudflare links. (*60.6*%)

tl;dr
### *39.4%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
