### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *162,187* public toots. (about *18020.8* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *68,612* links in total.
  - *17,232* links are Cloudflare links. (*25.1*%)
- There are *5,768* users who shared ANY links.
  - On average one mastodon user shared *11.9* links. (about *1.3* links per day)
  - There are *700* users who ONLY shared Cloudflare links. (*12.1*%)
  - There are *3,586* users who NEVER shared Cloudflare links. (*62.2*%)

tl;dr
### *37.8%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
