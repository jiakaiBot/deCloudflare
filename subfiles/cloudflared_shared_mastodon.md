### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *376,934* public toots. (about *17133.4* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *154,524* links in total.
  - *38,949* links are Cloudflare links. (*25.2*%)
- There are *8,467* users who shared ANY links.
  - On average one mastodon user shared *18.3* links. (about *0.8* links per day)
  - There are *962* users who ONLY shared Cloudflare links. (*11.4*%)
  - There are *4,924* users who NEVER shared Cloudflare links. (*58.2*%)

tl;dr
### *41.8%* of Mastodon users who shared link shared Cloudflare links.
### *25.2%* of shared links are Cloudflare links.
