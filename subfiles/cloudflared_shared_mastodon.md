### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *959,841* public toots. (about *17451.7* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *389,134* links in total.
  - *97,269* links are Cloudflare links. (*25*%)
- There are *12,987* users who shared ANY links.
  - On average one mastodon user shared *30* links. (about *0.5* links per day)
  - There are *1,327* users who ONLY shared Cloudflare links. (*10.2*%)
  - There are *7,043* users who NEVER shared Cloudflare links. (*54.2*%)

tl;dr
### *45.8%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
