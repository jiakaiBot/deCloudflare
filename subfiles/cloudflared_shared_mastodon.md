### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *694,885* public toots. (about *17372.1* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *279,237* links in total.
  - *70,046* links are Cloudflare links. (*25.1*%)
- There are *11,231* users who shared ANY links.
  - On average one mastodon user shared *24.9* links. (about *0.6* links per day)
  - There are *1,192* users who ONLY shared Cloudflare links. (*10.6*%)
  - There are *6,233* users who NEVER shared Cloudflare links. (*55.5*%)

tl;dr
### *44.5%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
