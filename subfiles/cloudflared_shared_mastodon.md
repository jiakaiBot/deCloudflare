### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *213,365* public toots. (about *17780.4* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *90,245* links in total.
  - *22,668* links are Cloudflare links. (*25.1*%)
- There are *6,569* users who shared ANY links.
  - On average one mastodon user shared *13.7* links. (about *1.1* links per day)
  - There are *778* users who ONLY shared Cloudflare links. (*11.8*%)
  - There are *3,987* users who NEVER shared Cloudflare links. (*60.7*%)

tl;dr
### *39.3%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
