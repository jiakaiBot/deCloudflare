### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *414,411* public toots. (about *17267.1* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *168,463* links in total.
  - *42,554* links are Cloudflare links. (*25.3*%)
- There are *8,843* users who shared ANY links.
  - On average one mastodon user shared *19.1* links. (about *0.8* links per day)
  - There are *986* users who ONLY shared Cloudflare links. (*11.2*%)
  - There are *5,120* users who NEVER shared Cloudflare links. (*57.9*%)

tl;dr
### *42.1%* of Mastodon users who shared link shared Cloudflare links.
### *25.3%* of shared links are Cloudflare links.
