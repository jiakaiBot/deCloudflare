### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *1,268,913* public toots. (about *17147.5* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *519,980* links in total.
  - *130,008* links are Cloudflare links. (*25*%)
- There are *14,629* users who shared ANY links.
  - On average one mastodon user shared *35.5* links. (about *0.5* links per day)
  - There are *1,466* users who ONLY shared Cloudflare links. (*10*%)
  - There are *7,800* users who NEVER shared Cloudflare links. (*53.3*%)

tl;dr
### *46.7%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
