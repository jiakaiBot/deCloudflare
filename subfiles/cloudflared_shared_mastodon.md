### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *836,809* public toots. (about *17433.5* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *337,176* links in total.
  - *84,245* links are Cloudflare links. (*25*%)
- There are *12,220* users who shared ANY links.
  - On average one mastodon user shared *27.6* links. (about *0.6* links per day)
  - There are *1,269* users who ONLY shared Cloudflare links. (*10.4*%)
  - There are *6,723* users who NEVER shared Cloudflare links. (*55*%)

tl;dr
### *45%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
