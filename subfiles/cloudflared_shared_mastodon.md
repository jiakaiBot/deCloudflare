### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *423,056* public toots. (about *17627.3* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *179,610* links in total.
  - *45,772* links are Cloudflare links. (*25.5*%)
- There are *8,898* users who shared ANY links.
  - On average one mastodon user shared *20.2* links. (about *0.8* links per day)
  - There are *962* users who ONLY shared Cloudflare links. (*10.8*%)
  - There are *5,183* users who NEVER shared Cloudflare links. (*58.2*%)

tl;dr
### *41.8%* of Mastodon users who shared link shared Cloudflare links.
### *25.5%* of shared links are Cloudflare links.
