### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *782,608* public toots. (about *17391.3* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *314,275* links in total.
  - *78,813* links are Cloudflare links. (*25.1*%)
- There are *11,847* users who shared ANY links.
  - On average one mastodon user shared *26.5* links. (about *0.6* links per day)
  - There are *1,234* users who ONLY shared Cloudflare links. (*10.4*%)
  - There are *6,552* users who NEVER shared Cloudflare links. (*55.3*%)

tl;dr
### *44.7%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
