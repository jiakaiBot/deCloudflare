### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *1,177,768* public toots. (about *17069.1* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *482,305* links in total.
  - *120,712* links are Cloudflare links. (*25*%)
- There are *14,199* users who shared ANY links.
  - On average one mastodon user shared *34* links. (about *0.5* links per day)
  - There are *1,423* users who ONLY shared Cloudflare links. (*10*%)
  - There are *7,608* users who NEVER shared Cloudflare links. (*53.6*%)

tl;dr
### *46.4%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
