### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *764,006* public toots. (about *17363.8* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *306,665* links in total.
  - *76,932* links are Cloudflare links. (*25.1*%)
- There are *11,733* users who shared ANY links.
  - On average one mastodon user shared *26.1* links. (about *0.6* links per day)
  - There are *1,234* users who ONLY shared Cloudflare links. (*10.5*%)
  - There are *6,507* users who NEVER shared Cloudflare links. (*55.5*%)

tl;dr
### *44.5%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
