### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *127,658* public toots. (about *18236.9* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *54,443* links in total.
  - *13,577* links are Cloudflare links. (*24.9*%)
- There are *5,194* users who shared ANY links.
  - On average one mastodon user shared *10.5* links. (about *1.5* links per day)
  - There are *658* users who ONLY shared Cloudflare links. (*12.7*%)
  - There are *3,257* users who NEVER shared Cloudflare links. (*62.7*%)

tl;dr
### *37.3%* of Mastodon users who shared link shared Cloudflare links.
### *24.9%* of shared links are Cloudflare links.
