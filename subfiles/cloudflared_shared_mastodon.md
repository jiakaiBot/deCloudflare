### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *676,579* public toots. (about *17348.2* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *271,329* links in total.
  - *67,991* links are Cloudflare links. (*25.1*%)
- There are *11,049* users who shared ANY links.
  - On average one mastodon user shared *24.6* links. (about *0.6* links per day)
  - There are *1,160* users who ONLY shared Cloudflare links. (*10.5*%)
  - There are *6,143* users who NEVER shared Cloudflare links. (*55.6*%)

tl;dr
### *44.4%* of Mastodon users who shared link shared Cloudflare links.
### *25.1%* of shared links are Cloudflare links.
