### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *108,838* public toots. (about *18139.7* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *46,127* links in total.
  - *11,446* links are Cloudflare links. (*24.8*%)
- There are *4,814* users who shared ANY links.
  - On average one mastodon user shared *9.6* links. (about *1.6* links per day)
  - There are *612* users who ONLY shared Cloudflare links. (*12.7*%)
  - There are *3,077* users who NEVER shared Cloudflare links. (*63.9*%)

tl;dr
### *36.1%* of Mastodon users who shared link shared Cloudflare links.
### *24.8%* of shared links are Cloudflare links.
