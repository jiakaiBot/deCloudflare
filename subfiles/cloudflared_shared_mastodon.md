### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *1,139,713* public toots. (about *17010.6* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *465,561* links in total.
  - *116,512* links are Cloudflare links. (*25*%)
- There are *13,986* users who shared ANY links.
  - On average one mastodon user shared *33.3* links. (about *0.5* links per day)
  - There are *1,425* users who ONLY shared Cloudflare links. (*10.2*%)
  - There are *7,480* users who NEVER shared Cloudflare links. (*53.5*%)

tl;dr
### *46.5%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
