### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *1,061,222* public toots. (about *17397.1* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *432,682* links in total.
  - *108,304* links are Cloudflare links. (*25*%)
- There are *13,590* users who shared ANY links.
  - On average one mastodon user shared *31.8* links. (about *0.5* links per day)
  - There are *1,384* users who ONLY shared Cloudflare links. (*10.2*%)
  - There are *7,300* users who NEVER shared Cloudflare links. (*53.7*%)

tl;dr
### *46.3%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
