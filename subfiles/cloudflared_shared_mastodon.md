### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *978,320* public toots. (about *17470* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *397,192* links in total.
  - *99,245* links are Cloudflare links. (*25*%)
- There are *13,101* users who shared ANY links.
  - On average one mastodon user shared *30.3* links. (about *0.5* links per day)
  - There are *1,332* users who ONLY shared Cloudflare links. (*10.2*%)
  - There are *7,103* users who NEVER shared Cloudflare links. (*54.2*%)

tl;dr
### *45.8%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
