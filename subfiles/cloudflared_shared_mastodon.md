### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *818,520* public toots. (about *17415.3* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *329,493* links in total.
  - *82,474* links are Cloudflare links. (*25*%)
- There are *12,092* users who shared ANY links.
  - On average one mastodon user shared *27.2* links. (about *0.6* links per day)
  - There are *1,266* users who ONLY shared Cloudflare links. (*10.5*%)
  - There are *6,651* users who NEVER shared Cloudflare links. (*55*%)

tl;dr
### *45%* of Mastodon users who shared link shared Cloudflare links.
### *25%* of shared links are Cloudflare links.
