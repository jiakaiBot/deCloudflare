### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *60,729* public toots. (about *15182.3* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *24,914* links in total.
  - *6,387* links are Cloudflare links. (*25.6*%)
- There are *3,301* users who shared ANY links.
  - On average one mastodon user shared *7.5* links. (about *1.9* links per day)
  - There are *420* users who ONLY shared Cloudflare links. (*12.7*%)
  - There are *2,210* users who NEVER shared Cloudflare links. (*66.9*%)

tl;dr
### *33.1%* of Mastodon users who shared link shared Cloudflare links.
### *25.6%* of shared links are Cloudflare links.
