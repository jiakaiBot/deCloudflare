### About Mastodon Users

[//]: # (do not edit this line start; t3)


- Mastodon users posted *251,109* public toots. (about *17936.4* toots per day)
  - Above number include toots which has no link.
- Now let's focus on *Mastodon toots which has links*.

![](../image/bully.jpg)

- Mastodon users shared *107,265* links in total.
  - *27,092* links are Cloudflare links. (*25.3*%)
- There are *7,039* users who shared ANY links.
  - On average one mastodon user shared *15.2* links. (about *1.1* links per day)
  - There are *819* users who ONLY shared Cloudflare links. (*11.6*%)
  - There are *4,225* users who NEVER shared Cloudflare links. (*60*%)

tl;dr
### *40%* of Mastodon users who shared link shared Cloudflare links.
### *25.3%* of shared links are Cloudflare links.
